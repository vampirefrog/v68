　・ＡＤＰＣＭの多重再生がリアルタイムで行える常駐型ドライバーです。

　・Ｘ６８０ｘ０全機種で動作し、

	1.１０ＭＨｚの機種でも８チャネル同時出力が可能である。
	2.チャネル毎に再生周波数および音量を指定することが出来る。
	3.細かい制御を行える専用ファンクションコールの他に、簡易使用のための
	　ＩＯＣＳコールを用意している。

　　などの特長を持っています。


-- 組み込み・使用法 -----------------------------------------------------------


　>PCM8 [option] で組み込まれます。option は以下の通り。
　オプションに誤りがある場合、簡易ヘルプが表示されます。

	-R		常駐解除
			他の常駐型アプリケーションによって占有されて
			いる場合は、解除出来ません。

	-D[num]		動作表示モード変更
			-D のみでトグル動作。
			-D0 でオフ、-D1 でオン。
			デフォルトはオフですが、オンにすると動作状態を
			視覚的に監視する事が出来ます。

	-V[num]		音量可変モード変更。
			-V のみでトグル動作。-V0 でオフ、-V1 でオン。
			標準設定はオンです。
			オフにすると、以後の出力音量は 8(原音) に固定
			されます。

	-Inum		IOCS で使用する最大チャネル数の指定。
			num は 0〜8 で、標準設定は 8 です。
			通常は多重再生状態の場合のみ意味を持ちますが、
			0 にすると、単音・多重に拘わらず、IOCS 出力が
			禁止されます。

	ON		多重再生を許可する
			デフォルトは、このモードです。

	OFF		多重再生を禁止する。

	FNC		専用ファンクションコールのみ、多重再生を許可する。
			IOCS コールは単音再生になります。

　-R 以外は常駐中の設定変更も可能です。

　　バグ取りを中心に行いたいと考えています。POLYPHON 版の PCM8SB はループ
　　音階など POLYPHON 独自の機能への対応が残されておりますので、こちらは
